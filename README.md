# 3D桌球大师游戏项目说明文档

## 🎮 项目概述
这是一个基于Web技术栈开发的3D桌球游戏，支持单人练习、多人在线对战和完整的用户系统。游戏具有精美的3D图形、流畅的物理引擎和稳定的网络对战功能。

## 📁 项目文件结构

```
billiard-game/
├── index.html                      # 2D原版游戏主页面
├── config.js                       # 2D游戏配置文件
├── style.css                       # 2D游戏样式
├── game.js                         # 2D游戏主逻辑
├── 部署指南.md                     # 原版部署文档
├── 3d-billiard-game.html          # 3D新版游戏主页面
├── 3d-billiard-game.js            # 3D游戏主逻辑（客户端）
├── 3d-billiard-server.js          # 3D游戏服务器代码（服务端）
└── README.md                       # 项目说明文档
```

## 🚀 部署指南

### 3D游戏快速部署

#### 客户端部署
1. 直接在浏览器打开 `3d-billiard-game.html` 即可运行游戏
2. 如需远程部署，可将HTML文件上传到任何静态网页托管平台

#### 服务端部署（可选，用于多人对战）
```bash
# 1. 安装依赖
npm install express socket.io cors bcrypt jsonwebtoken mongoose

# 2. 启动服务器
node 3d-billiard-server.js

# 3. 修改客户端连接地址
# 在3d-billiard-game.js中修改socket连接地址
this.socket = io('https://your-server.com');
```

### 测试连接
1. 在不同浏览器或设备中打开游戏
2. 以游客模式登录
3. 点击"匹配对战"
4. 等待匹配后开始游戏

## 🎮 游戏操作说明

### 基本操作
1. **触摸操作**：触摸白球拖动来瞄准和击球
2. **力度控制**：拖动距离决定击球力度
3. **方向控制**：拖动角度决定击球方向

### 对战流程
1. 玩家A击球 → 等待球停止
2. 玩家B击球 → 等待球停止
3. 循环，直到一方获胜

### 胜负判定
- 进球数量更多的一方获胜
- 白球进袋会被重置位置
- 非法击球会被判负

## 📊 性能优化

### 渲染优化
- 使用阴影贴图优化阴影效果
- 材质参数调节以平衡画质和性能
- 使用视锥体剔除减少渲染负载

### 物理优化
- 调整物理引擎迭代次数
- 降低不必要的碰撞检测
- 优化刚体运动参数

### 网络优化
- 二进制协议减少带宽占用
- 本地预测降低延迟
- 断线重连机制

## 🛠️ 常见问题

### 开发环境问题
1. **游戏不显示**：确保浏览器支持WebGL
2. **运行卡顿**：尝试降低图形质量设置
3. **网络连接失败**：检查服务器地址是否正确

### 游戏功能问题
1. **无法匹配对手**：当前可能没有其他在线玩家
2. **操作不响应**：请等待球完全停止
3. **球会穿墙**：这是物理引擎模拟误差，会自动修正
